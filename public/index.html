FCC Image Abstraction Layer

Usage:

To search for images, call this url with /api/imagesearch/yoursearchhere?offest=0

Search will return a json object with descriptive text, thumbnail url, and page url where the image is located.

You can page through the results by seeing offset to the page you want to view.

View Search History

To view a history of search results, call this url with /api/lastest/imagesearch
